About

I work at the intersection of cloud infrastructure, security, and developer experience. My focus is on building systems that are secure by default, automated where it counts, and simple enough that teams actually want to use them.

I've spent years in the Azure ecosystem, from cloud architecture and DevOps pipelines to threat modeling and security tooling. More recently, I've been deep in the generative AI space, exploring how LLMs can augment engineering workflows, improve security posture, and create new kinds of developer tools.

When I'm not building, I'm usually playing video games with my kids or tinkering with something that probably doesn't need tinkering with.
